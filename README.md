# AoC 2022

Project for the Advent of Code 2022
Adonisjs server
Vuejs client

## Setup

Clone the repo

```bash
git clone git@github.com:opmvpc/aoc22.git
```

Install dependencies

```bash
npm install
```

## Run in dev mode

```bash
npm run dev
```

## Build for production

```bash
npm run build
```

# AoC 2022

Project for the Advent of Code 2022

- Adonisjs server
- Vuejs client

## Setup

Clone the repo

```bash
git clone git@github.com:opmvpc/aoc22.git
```

Install dependencies

```bash
npm install
```

## Run in dev mode

```bash
npm run dev
```

then open `http://127.0.0.1:5173/front/`

## Build for production

```bash
npm run build
```

## Run tests

```bash
npm run test
```
